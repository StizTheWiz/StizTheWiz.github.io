Vibrant Venture

Step 1: Download the game here.

Step 2: When the download is finished, click "show in folder."

Step 3: Move the "Vibrant_Venture" folder out of Downloads and into the Local Disk (C:). The easiest way to do this is to click and drag the file to the local disk button on the left hand side. You could also right click the folder, select copy, and paste into the local disk.

Step 4: Right click the folder, then select "extract all." When the pop-up appears, ignore the text and click "extract." Once the files have been extracted, you can delete the zipped folder if you want to.

Step 5: Open the newly extracted "Vibrant_Venture" folder, then double click the "Vibrant Venture" file. You're in!

Tips:

  • You can leave this game running in the background if you so choose (although it will use up a lot of battery).
  • To close this game, you can either click "quit" on the main menu or just close the window.
  • To find the game again, select This PC on your desktop, then select Local Disk (C:). This game will NOT be removed by shutting down your laptop, but will be removed if the laptop is reimaged.
  • Unfortunately, this is just a demo, so there are only three levels. However, that should be enough content to last a while (especially if you go for 100%)
